AquaPrivate (offline Maven)

Сборка:
1) Убедитесь что файлы лежат по пути:
   C:\1.21.8\spigot-api-1.21.8-R0.1-SNAPSHOT.jar
   C:\1.21.8\worldedit-bukkit-7.3.16.jar
   C:\1.21.8\worldguard-bukkit-7.0.14-dist.jar

2) В папке проекта выполнить:
   mvn -o package

3) Готовый jar будет в:
   target\AquaPrivate-1.0.0.jar
